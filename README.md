## 🧠 Sobre o Projeto

API REST desenvolvida com **Java + Spring Boot** para gerenciamento de cursos de programação.

O projeto simula uma aplicação onde é possível realizar operações completas de CRUD (Create, Read, Update, Delete) em cursos.

---

## 🎯 Funcionalidades

- Criar um novo curso
- Listar todos os cursos
- Filtrar cursos por nome e categoria
- Atualizar um curso pelo ID
- Remover um curso pelo ID
- Alternar status ativo/inativo do curso

---

## 🏗️ Estrutura da Entidade

| Campo      | Descrição |
|-----------|----------|
| id        | Identificador único |
| name      | Nome do curso |
| category  | Categoria do curso |
| active    | Indica se o curso está ativo |
| createdAt | Data de criação |
| updatedAt | Data de atualização |
